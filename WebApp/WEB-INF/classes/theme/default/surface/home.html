    February 23, 2016— 0 comments

    中国证券网讯（记者 周鹏峰）中国银行(3.550, -0.03, -0.84%)25日发布2016年11月跨境人民币指数（CRI），结果显示，11月CRI指数为251点，较上月上升3点，较上年末下降25点。
